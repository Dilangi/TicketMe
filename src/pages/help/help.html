About Us

TicketMe is a train ticket purchasing mobile application for selected trains. You donâ€™t need to go to collect tickets from any stations. You can use the QR code of your purchased ticket as a virtual ticket.

Ticket purchasing process:

  • Sign-up for the application
  • Sign in to the application
  • Select start station, destination, class information and number of tickets
  • Buy tickets using mobile credits or credit/debit card

Available Destinations:

  • Colombo-Kandy
  • Colombo-Badulla
  • Colombo-Jaffna
  • Colombo-Batti
  • Colombo-Trinco
  • Colombo-Matara

Features:


24 hour access, No queuing, No travelling. Ticket charges will be charged for every successful booking. These fees will be added to your ticket price.

Ticket Price:


Ticket fee + Purchasing charge + Taxes would be applicable



Vision

Build an alternative and efficient way to mitigate the drawbacks of the current ticketing system and increase efficiency and convenience of buying tickets through the mobile application.



Limitation

TicketMe is for daily ticket buying and monthly private season buying purpose and not for ticket booking function. Considering about the payment method, it can be done by credit card or Mobitel post-paid sim card. However the number of QR codes creates at a time is limited, therefore it could be occure an inconvinience.



Our Customers

TicketMe is an servise oriented application for daily ticket users and monthly private season users.



Obligations, Liabilitias and Risks

  • Data loss are mitigated by keeping multiple backups of system and database in multiple locations.
  • Security Issues are mitigated by ensuring that the server has considerable security measures.
  • Passenger information are secured by providing different access levels to different layers of user and provide capability to keep different passwords for every user login.
